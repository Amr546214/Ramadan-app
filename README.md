# 🌙 Ramadan React App

A modern React application combining Islamic features with interactive UI.

## ✨ Features
- Ramadan Quiz System
- Prayer Times API integration
- Next Prayer Detection Algorithm
- Real-time Clock
- Morning Azkar with Counter
- Browser Notifications
- Adhan Sound

## 🛠 Tech Stack
- React
- JavaScript
- CSS
- REST API
- SweetAlert2
